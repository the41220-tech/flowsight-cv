> 관점: CV 엔지니어 + 사업 전략 분석가

> 검증 상태: **테스트 26/26 라이브 재실행 통과**(moat2 12 + anomaly 10 + wildtrack 4), 핵심 모듈 전부 존재 확인, 모든 수치 프로젝트 로그·세션 결과와 대조 완료.

## 1. 요약

FlowSight AI는 입력원에 무관한(드론/CCTV/거치형) 실시간 **군중-움직임 분석 CV 플랫폼**으로, 사람을 2D/3D 메트릭 지도에 복원하고 밀도·흐름장을 만들어 이상 군중 동역학을 탐지·해석한다. 압사 예측은 다수 응용 중 하나일 뿐이다.

핵심 **해자**는 경쟁사의 평면 호모그래피 히트맵과 달리 **비평면 3D 메트릭 측위 + 지형 중력 위치에너지/압력장**(U=ρgh + ∇U)을 **Helbing 군중압력** P=ρ·Var(v)[1/s²]과 결합한 것이다. 이 해자는 합성·단위 검증을 넘어 이번 세션에서 **실제 측량 캘리브레이션 데이터(WILDTRACK)에서 기하 정확성이 0.0000m로 입증**되었다.

전체 하드테크는 **~70%**, 물리 해자는 **~90%**(실데이터 검증 포함) 수준이다.

## 2. 성과 종합 (영역별 · 핵심 수치)

### 인지 (Perception)
- **FT-1**(YOLO11m, VisDrone): 드론 recall 0.154→**0.512**(3.3×), 그러나 CCTV 망각 0.685→0.117.
- **FT-2**(혼합 30ep): 드론 **0.571** · CCTV **0.455**(망각 회복) · 소형 **0.517** · 전체 **0.552** · 정밀 **0.612**, VisDrone-val mAP50 **0.60**.
- **SAHI**(무재학습): 드론 0.571→**0.712**(+25%) · 소형 **0.673**(+30%) · 전체 **0.698**.
- 멀티모델 벤치: rtdetr 0.573 / 앙상블 **0.642**. 개방어휘 장애물 모듈.

### 기하 (Geometry)
- 보정기 3종: Homography(4점+해석 야코비안), Pedestrian(1.7m), **DepthGround(메트릭깊이→지면→미터, 무측량)** + 지형 DEM + 멀티캠 융합 + WILDTRACK 실캘리브 로더.
- **H1**: 메트릭깊이 역투영이 경사 지형 수평오차를 단일 호모그래피 대비 **−78%**(광선교차 −73%); 단순 다중 호모그래피는 반증.

### 물리 해자 (Physics moat)
- 절대 압력 frame_pressure_metric(→1/s²) + alarm_level(**0.02 절대 3단계**), U=ρgh, MoatMonitor / DisasterMonitor.
- **H2**: 지형 위치에너지 전조가 압사 **+7.2s** 선행, Helbing 난류보다 **18.8s** 앞섬.
- **H8**: 단위 **7/7**(ρ=4·Var=0.18 → P=0.72/s² 정확). 합성 E2E에서 절대 0.02/s² 경보가 압사 **+10.47s** 선행(3시드), 최대 P 임계의 **9–87×**, 최대 밀도 **7.3–10.1/m²**(이태원 실측 7.57).

### 이상탐지 스위트 (Anomaly suite)
- 방사발산 · 고속접근 · 비상공백 · 지오펜스 · 테러 컴포지트 · 폭력(학습) · 낙상(자세) · 한국어 narrator · 히스테리시스 게이트. **테스트 26/26**.
- **H6**: 멀티모달 경사→재난이 압사 **5.2–6.0s** 선행, 검출 recall **30–51%**에서도 ~5–6s 유지.
- **H7b**: 저 recall에서 밀도 경보는 압사를 누락, 멀티모달 해자는 무관하게 조기 선행.

### 멀티카메라 (WILDTRACK 실데이터, 최신)
- EPFL 6.8GB zip 손상(>4GB 중앙디렉토리 오프셋 래핑 → unzip/7z/zipfile 전부 실패) → **자체 순차 로컬헤더 파서**로 해결: C1=401 + C2=401 프레임 + 캘리브 21 + 주석 400 추출(손상 0).
- **해자 기하 실측 캘리브에서 정확**: 발끝→지면 라운드트립 오차 **0.0000m**, GT positionID→월드가 공식 **12×36m** 평면과 정확 일치, GT 재투영 **100% in-image**.
- 융합(근지평선 클램프 + 카메라↔폴더 자동매핑): 1캠 recall **0.085** → 2캠 **0.102**(TP **54→65**), 위치오차 **0.60m** → **실데이터에서 융합이 가림을 메워 검출↑(2캠>1캠)**.
- 실지면 절대 밀도: GT peak **1.05–1.27/m²**(물리적 타당).

### 실데이터 런 (기타)
- VisDrone 실프레임 → RT-DETRv2 46명 → BEV. 폭력 분류기 RWF-2000(2000클립 균형) val top1 **~0.86**. UMN 패닉 실영상: 흐름-압력 경보가 **7개 에피소드 전부** onset 선행(+1.6–4.3s, 평균 ~3.9s). ByteTrack(FT-2) **117–433 ID** + 인당 x,y,vx,vy + 체류. 대시보드 v2. MOAT-2 Colab L4 절대압력 실 1/s², 실프레임 메트릭깊이 4.9–59.4m.

## 3. 실데이터 검증 vs 합성/단위 검증 (정직한 구분)

| 실데이터에서 입증 | 합성/단위테스트로 입증 |
|---|---|
| WILDTRACK 기하: 라운드트립 **0.0000m**, 12×36m 정확 일치, 재투영 100% | H8 합성 E2E 경보 선행 **+10.47s**(3시드) |
| WILDTRACK 융합: 1캠 0.085 → 2캠 0.102(TP 54→65), 위치오차 0.60m | H8 단위 7/7(절대단위 산술 정확) |
| WILDTRACK 절대 밀도: GT peak 1.05–1.27/m² | H1 경사오차 −78%/−73%(합성 지형) |
| UMN 패닉: 7/7 에피소드 onset 선행(평균 3.9s) | H2 지형 전조 선행 +7.2s/+18.8s(사회력 시뮬) |
| FT-1/FT-2/SAHI recall(실 VisDrone/COCO) | H6 멀티모달 선행(검출 시뮬 결합) |
| 폭력 분류기 top1 ~0.86(RWF-2000 실클립) | |

**정직한 한계**: WILDTRACK 절대 엔드투엔드 recall ~**10%**. 원인은 밀집 군중에서 **COCO 발끝-투영을 7뷰 중 2뷰만** 사용한 검출 방식(SOTA 학습형 멀티뷰 ~90%)이며, **해자 기하의 문제가 아니다**(기하는 0.0000m로 정확). 즉 약점은 교체 가능한 상품화 구간(검출기)에 있다.

## 5. 전략적 결론

- **해자는 물리(비평면 절대 스케일)이며, 실데이터에서 정확히 작동한다**(0.0000m). 경쟁사의 평면 히트맵이 복제할 수 없는 부분이 검증됐다.
- **약점은 검출 프런트엔드**(상품화 구간) — 학습형 멀티뷰 검출기로 교체하면 recall이 오르며, 이는 명확하고 분리된 다음 투자처다. 멀티카메라 융합(E)이 단일시점 사각을 제거하는 **차기 차별점**이고, 이를 가능케 하는 per-camera 메트릭 보정은 이미 확보됨.
- 권장 순서: **E(멀티캠 하드닝, 핵심 해자) → F(외적검증·경보 정밀도) → D의 VLM 설명 → 엣지(v3)**. recall 레버(SAHI/FT-3)는 기회 시 병행.

> 이번 세션의 추가 의의: 외부 장벽(손상 zip·unzip 불가·런타임 끊김)을 자체 순차 파서·이어받기·재연결로 모두 우회하여 실데이터 검증을 완수.
